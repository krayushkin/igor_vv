Разработка на языке Си набора процедур реализации функции для случая использования данных типа float
----------------------------------------------------------------------------------------------------

.. include:: float.c
   :literal:
   :start-after: //start FlFunc
   :end-before: //end FlFunc

Требуемая точность достигается для x в диапазоне от 0 до 0,8. Для этого используется ряд из 11 слагаемых.
При меньшей длине ряда требуемая точность не достигается.

1. ``FlMath``. Релизация с использованием функций из библиотеки math

   Используется tanf(x) из стандартной библиотеки.

2. ``FlCyclNoGorner``. Прямая реализация с использованием цикла

   Для оптимизации по скорости для вычисления степеней x.

3. ``FlNoCyclGorner``. Безцикловая реализация по схеме Горнера

   Оптимизация: заранее вычисляется значение x*x.

4. ``FlCyclGorner``. Реализация по схеме Горнера с циклом

   Оптимизация: заранее вычисляется значение x*x.

5. ``FlNoCyclNoGorner``. Прямая реализация без цикла
